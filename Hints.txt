___________________________________________________________________

Search maybe 3 dayes a failure. It is embarrassing

would use a node structure for XML::LibXML. Thougt a i have to install some more compontents 

But nothing works. Perl doesn't know functions like "hasChildNodes()"

So i changed my parser initialization, it looks like: 

Doesn't work
$test = $parser->parse_string($fileContent{$key});
$root = $test->getDocumentElement();	
$node = $root->hasChildNodes();

Work
$test = $parser->parse_file("test.xml");
$root = $test->getDocumentElement();	
$node = $root->hasChildNodes();
